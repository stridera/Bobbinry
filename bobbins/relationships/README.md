# Relationships

Graph and matrix views for entity connections across any bobbin. Reads
the relationship metadata that other bobbins (entities, manuscript,
timeline) record on their entities, then renders the network so you can
see who knows whom.

- `shell.leftPanel` panel for navigation between graph / matrix views.
- Bobbin-agnostic: queries entities by relationship metadata, not by type.
- Pure UI — no external services.
