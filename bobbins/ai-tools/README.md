# AI Tools

AI Tools is an **analysis and suggestion tool**. It does not write, rewrite, or generate content for you.

## Philosophy

Bobbinry's stance on AI is clear: **tools that write content for users are not allowed.** AI Tools exists solely to help writers analyze and improve their own work — the same way a beta reader or writing coach would.

Nothing runs automatically. Nothing modifies your work without explicit action.

## Features

### Synopsis Suggestions

Suggests a short summary based on your chapter content. You review the suggestion, edit it if you like, and decide whether to save it. AI-assisted synopses are tagged with provenance metadata so the origin is always transparent.

### Structured Feedback

Provides structured feedback like a beta reader — covering pacing, character voice, prose quality, and actionable suggestions. Reviews are generated on demand and are not persisted.

## Setup

1. Install the AI Tools bobbin on your project
2. Open the AI Tools panel (right sidebar)
3. Enter your own API key (Anthropic Claude or OpenAI)
4. Open a chapter and use the Generate buttons

## Privacy

- Your API key is stored in your account settings, never shared
- Chapter content is sent to the AI provider you choose, only when you click Generate
- No content is stored by the AI provider beyond the API call
