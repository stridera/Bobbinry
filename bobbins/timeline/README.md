# Timeline

Chronological event tracking for plot and world history. Use it for
in-story timelines (kingdoms rising and falling) or out-of-story
production timelines (when chapter X happens relative to chapter Y).

- `shell.leftPanel` panel for navigation between timelines and events.
- Custom event view supports flexible date scales (days, years, eras).
- Stores timelines + events as entities in its own collection.
