# Manuscript

Complete manuscript management with books, chapters, and scenes. The
canonical writing surface in Bobbinry — most projects install this first.

- `shell.leftPanel` panel with the project navigation tree (books → chapters → scenes).
- `shell.editorFooter` view with live session stats (words written this session, etc.).
- Publishable: chapters from this bobbin can be picked up by `web-publisher`.
- AI-aware: integrates with the `ai-tools` bobbin for synopsis and review.
